# Stacks Builder Challenge 

##  Challenge Overview
This repository contains **30 innovative smart contract projects** built for the Stacks Builder Challenge. Each project demonstrates different **Clarity features** and showcases advanced Bitcoin L2 development patterns across DeFi, NFT, Gaming, DAO, Infrastructure, and more.

##  Quick Start

### Install Dependencies
```bash
# Install Clarinet
npm install -g @hirosystems/clarinet

# Verify installation
clarinet --version
```

### Check All Projects
```bash
# Navigate to each project and run
cd digital-will && clarinet check
cd ../secure-escrow && clarinet check
cd ../contract-verifier && clarinet check
cd ../btc-bridge-helper && clarinet check
cd ../passkey-auth && clarinet check
```

### Deploy to Testnet
```bash
# From each project directory
clarinet deploy --testnet
```
